# 📊 Motorola Sales Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Sales Analytics Dashboard** developed in **Microsoft Power BI** to analyze Motorola's sales performance across different cities, mobile models, payment methods, and time periods.

The dashboard provides business insights through interactive visualizations, KPIs, filters, and drill-down capabilities, enabling users to explore sales trends and customer purchasing behavior.

## 📊 Dashboard Components

### 🗺️ Total Sales by City
Displays sales distribution across different cities using an interactive map.

### 📈 Monthly Units Sold
Shows month-wise sales trend to identify seasonal patterns.

### ⭐ Customer Rating Analysis
Visualizes customer satisfaction based on product ratings.

### 💳 Payment Method Distribution
Analyzes customer payment preferences including:
- UPI
- Cash
- Debit Card
- Credit Card

### 📱 Mobile Model Performance
Compares sales across different Motorola smartphone models.

### 📅 Day-wise Sales Analysis
Displays sales performance based on the day of the week.

### 📋 Sales Summary Table
Provides detailed sales statistics for each mobile model including:
- Quantity Sold
- Total Sales
- Total Transactions
